num = int(input('digite um número')) if num % == 0; print(f'o número {num} é par') else: print(f'o número {num} é impar') print('fim do programa')

quando escrevemos apenas desta forma a cima, fica tudo de forma contínua.

porém podemos utilizar a tag "pre" e fica tudo da forma que está escrito:

num = int(input('digite um número'))
        if num % == 0;
        print(f'o número {num} é par')
        else:
        print(f'o número {num} é impar')
        print('fim do programa')

inclusive, quebras de linhas (enter) são considerados quando estão dentro da tag "pre"
